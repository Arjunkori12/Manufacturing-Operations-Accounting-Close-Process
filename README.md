# Manufacturing Operations Accounting & Close Process

## Overview
This project simulates a full manufacturing accounting and month-end close process for a steel production facility. Built in Microsoft Excel, the model reflects real-world operational accounting, internal controls, and management reporting.

## Key Features
- Month-end journal entries with debit/credit validation  
- Bank reconciliation with automated balance checks  
- Inventory aging and obsolescence reserve calculations  
- GRIR and Accounts Payable reconciliation  
- Freight expense analysis by carrier and cost per lb  
- Weekly spending tracking and trend analysis  
- Budget vs actual variance reporting  
- Quarterly forecast with scenario analysis  

## Tools Used
- Microsoft Excel (Advanced formulas, Pivot Tables, data validation)
